# Dumbathon Apocalypse Website

Welcome to the Dumbathon Apocalypse Command Center! This website is designed to be a chaotic, immersive experience representing a survivor terminal during an apocalypse.

## How to Use the Website

1.  **Navigation**: There is no traditional navigation bar. You interact with the site through the 'Apocalypse Command Center' dashboard on the main screen.
2.  **Global Transmission Feed**: View the latest chaotic updates from around the wasteland.
3.  **Survivor Console**: Access the terminal to run commands and interact with the system. Type commands into the prompt.
4.  **Security Gate**: A friction-based entry point. You may need to solve puzzles, wait, or furiously click to gain access.
5.  **Broadcast**: Listen to the static and hidden messages.
6.  **Leaderboard / Survivor Rankings**: See who has survived the longest or gathered the most scrap.
7.  **Chaos System**: Be prepared for random visual glitches, system alerts, and UI elements shifting around. Do not panic—this is by design!

## Local Development

If you want to run this locally:

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Run the development server: `npm run dev`
4.  Open [http://localhost:3000](http://localhost:3000) in your browser.
